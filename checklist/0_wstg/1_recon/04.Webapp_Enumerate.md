# 🔍 WEB SERVER APPLICATION ENUMERATION CHECKLIST

 ## Comprehensive Web Application Enumeration

### 1 Directory and File Enumeration
    - Common Directory Discovery:
      * Administrative interfaces: /admin, /administrator, /manager
      * Application directories: /app, /application, /src
      * Configuration directories: /config, /settings, /conf
      * Backup directories: /backup, /backups, /bak
      * Log directories: /logs, /temp, /tmp

    - File Extension Brute Forcing:
      * Configuration files: config, conf, ini, xml, json
      * Backup files: bak, backup, old, tmp, save
      * Source code: php, jsp, asp, aspx, py, rb
      * Database files: sql, db, mdb, sqlite

    - Automated Directory Brute Forcing:
      * Gobuster: gobuster dir -u http://target.com -w wordlist.txt
      * Dirb: dirb http://target.com /usr/share/dirb/wordlists/common.txt
      * FFuF: ffuf -u http://target.com/FUZZ -w wordlist.txt
      * Dirsearch: dirsearch -u http://target.com -e php,html,js
